WR1

Explain the difference between a Series and a DataFrame.

Expected answer:
A Series is one-dimensional; a DataFrame is two-dimensional.

WR2

Why is it important to check for missing values before analysis?

Expected answer:
Missing values can distort calculations and conclusions.

WR3

Write Pandas code to filter rows where revenue is greater than 5,000.

Expected answer:

df[df["revenue"] > 5000]

WR4

Write code to group data by region and calculate total revenue.

Expected answer:

df.groupby("region")["revenue"].sum()

They create new insights from existing data.
